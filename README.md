# The Art Loom

# ARTISAN MARKETPLACE PROJECT

![](image/logo.jpg)

The **Art Loom** is a specialized marketplace platform designed to promote and trade handcrafted goods and traditional artisan services. It focuses on regional heritage, allowing users to discover unique items like Channapatna toys, hand-woven textiles, and local art.

### Developer
- **Sanjana Angadi**

## Project Overview
In this project, I have developed a functional e-commerce clone tailored for the artisan community. The goal was to create a seamless user experience for browsing regional crafts while implementing core web technologies.

## Tech Stacks Used
1. **Vanilla JavaScript** - Handles the dynamic dropdown logic, state selection, and language switching.
2. **HTML5** - Provides the structural foundation for the home and category pages.
3. **CSS3** - Custom-built styling to create the Art Loom brand identity and layout.

## Key Features
1. **State-Based Filtering:** A custom-coded dropdown allowing users to filter crafts by their specific region or state.
2. **Rebranded Interface:** A complete transformation of the UI to reflect an artistic and cultural marketplace.
3. **Search Functionality:** An integrated search bar to find specific products like "Wooden Toys" or "Embroidery."
4. **Multilingual Support:** Implementation of language selection (English and Kannada) for better local accessibility.
5. **Product Categorization:** Organized layouts for various craft types, from home decor to traditional fashion.

## Project Structure
* `index.html` - Main landing page.
* `category.html` - Product browsing and filtering page.
* `script/index.js` - Logic for UI interactions and state management.
* `style/category.css` - Global and page-specific styling.
* `image/` - Directory for all brand logos and product imagery.

## Future Scope
* **Mobile Responsiveness:** Adapting the layout for mobile and tablet devices.
* **Backend Integration:** Connecting to a live database for real-time inventory management.
